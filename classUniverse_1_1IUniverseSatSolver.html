void IUniverseSatSolver::addAllClauses ( const std::vector< std::vector< int >> &  clauses)
virtual

Adds to this solver several clauses from a set of sets of literals. This is convenient to create in a single call all the clauses.

Parameters
clausesA vector of sets of literals in the DIMACS format. The vector can be reused since the solver is not supposed to keep a reference to that vector.
Exceptions
UniverseContradictionExceptionIf one of the vectors of literals is empty or contains only falsified literals after unit propagation.

virtual void Universe::IUniverseSatSolver::addClause ( const std::vector< int > &  literals)
pure virtual

Adds to this solver a clause from a set of literals. The literals are represented by non-zero integers such that opposite literals are represented by opposite values (following the classical DIMACS way of representing literals).

Parameters
literalsA set of literals.
Exceptions
UniverseContradictionExceptionIf the vector of literals is empty or if it contains only falsified literals after unit propagation.
