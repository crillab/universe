The latest release is available \href{https://github.com/crillab/universe/releases/latest}{\texttt{ here}}

{\ttfamily UNIVERSE} is developed using \href{https://en.cppreference.com/w/cpp/17}{\texttt{ C++ 17}}, \href{https://cmake.org/}{\texttt{ CMake 3.\+23}}. Installing CMake is required if you want to build from source. To do so, after having installed all the needed tools, you will need to clone the project\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{git clone https://github.com/crillab/universe.git -\/-\/recursive}
\DoxyCodeLine{cd universe}
\DoxyCodeLine{mkdir build}
\DoxyCodeLine{cd build }
\DoxyCodeLine{cmake .. -\/DCMAKE\_BUILD\_TYPE=RELEASE}
\DoxyCodeLine{cmake -\/-\/build . }

\end{DoxyCode}
\hypertarget{md__home_runner_work_universe_universe_doc_INDEX_autotoc_md3}{}\doxysection{Using CMake}\label{md__home_runner_work_universe_universe_doc_INDEX_autotoc_md3}
This library is designed for simplified use in projects using CMake. To use this project, it is possible to create a sub-\/module in your project with the following commands\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{git submodule add https://github.com/crillab/universe libs/universe}
\DoxyCodeLine{git submodule update -\/-\/remote -\/-\/recursive -\/-\/init}

\end{DoxyCode}


And then add the following instructions in your CMake\+Lists.\+txt file\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{add\_subdirectory(libs/universe)}
\DoxyCodeLine{add\_executable(your\_solver solver.h solver.cpp)}
\DoxyCodeLine{target\_link\_libraries(your\_solver universe)}

\end{DoxyCode}
